## Details of dataset available for Face Detection are below

| Dataset Name | URL    | Definition    |
| :---:   | :--- | :--- |
| PASCAL Face | [Data](https://www.kaggle.com/datasets/gopalbhattrai/pascal-voc-2012-dataset)   | Subset of PASCAL data   |
| Wider Face | [Data](http://shuoyang1213.me/WIDERFACE/)   | WIDER FACE dataset is a face detection benchmark dataset, of which images are selected from the publicly available WIDER dataset.   |
| AFLW | [Data](https://web.archive.org/web/20170513171430/http://lrs.icg.tugraz.at/research/aflw/)   | Annotated Facial Landmarks in the Wild   |
| MAFA | [Kaggle Data](https://www.kaggle.com/datasets/revanthrex/mafadataset)   | Masked Faces  |
